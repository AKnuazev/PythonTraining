# Google Play Store parsing app
## Required:
- json
- requests
- os
- bs4
- selenium
- threading
- time
- math

## Apps information displayed:
- name
- url of the application page
- author
- category
- description
- average rating
- number of ratings
- Last update

## Implementation features:
- Accessing a page using a browser
- Special tools for parsing pages
- Multithreading
